# Chrome Web Store Listing

## Extension Name
ShopifySEO Pro - Free SEO Analyzer

## Short Description (max 132 characters)
Free Shopify SEO analyzer with 50+ checks, Core Web Vitals, site crawler, rank tracker, and competitor comparison. No API needed.

## Detailed Description
ShopifySEO Pro is the most comprehensive free SEO analysis tool built specifically for Shopify stores. Run a complete SEO audit directly in your browser — no account, no API keys, no monthly fees.

★ 50+ SEO CHECKS ACROSS 7 CATEGORIES
Analyze Meta tags, Content quality, Technical SEO, Images, Links, Performance, and Shopify-specific issues in one click.

★ REAL CORE WEB VITALS
Not estimates — actual LCP, CLS, FCP, and TTFB measurements from the browser's Performance API. Know exactly how fast your store loads.

★ FULL SITE CRAWLER
Discover all pages via sitemap.xml, batch analyze up to 50 pages, and find the most common SEO issues across your entire site.

★ GOOGLE RANK CHECKER
Check where your store ranks for any keyword. Track position changes over time. See exactly who's outranking you.

★ COMPETITOR COMPARISON
Paste any competitor URL for an instant side-by-side SEO comparison. See what they do better and where you're winning.

★ INTERNAL LINK MAP
Visual graph of your internal link structure. Find orphan pages, see link distribution, and identify weak spots.

★ COPYABLE FIX CODE
Every issue comes with ready-to-paste Shopify Liquid code snippets. Click "Copy" and paste directly into your theme files.

★ PDF REPORTS & CSV EXPORT
Generate professional PDF reports for clients. Export detailed CSV spreadsheets with all issues and recommendations.

★ SHOPIFY-SPECIFIC CHECKS (15+)
- Product pages: description quality, image count, reviews, variant canonicals
- Collections: description presence, tag page indexation
- Blog posts: word count, article schema, internal linking
- Theme: breadcrumbs, search, app bloat detection, hero image optimization

★ EDITABLE SERP PREVIEW
See exactly how your page appears in Google. Edit the title and description live with character counters.

★ SCORE HISTORY
Track your SEO score improvements over time. See trends and score changes for every domain you analyze.

100% FREE — No accounts, no API keys, no data collection. Everything runs locally in your browser. Your data never leaves your device.

Works on any website, with special Shopify-specific analysis when a Shopify store is detected.

Screenshot Guide for Chrome Web Store

Chrome Web Store requires screenshots at 1280x800 or 640x400 pixels. You need 1-5 screenshots.

Step 1: Set up your browser

  1. Open Chrome DevTools (F12)
  2. Click the device toolbar icon (or Ctrl+Shift+M)
  3. Set dimensions to 1280 x 800
  4. Navigate to a Shopify store (e.g. allbirds.com, gymshark.com, fashionnova.com)

Step 2: Take these 5 screenshots

For each, open the ShopifySEO Pro side panel and capture the full panel.

  1. Screenshot 1 - Overview Dashboard
    Run analysis, show the Overview tab with score gauge, category bars, and top issues.
    Save as: screenshot-1-overview.png
  2. Screenshot 2 - Meta & SERP Preview
    Click Meta tab, show the editable SERP preview with character counters and social preview.
    Save as: screenshot-2-meta.png
  3. Screenshot 3 - Technical + Structured Data
    Click Technical tab, expand a schema block to show the field viewer.
    Save as: screenshot-3-technical.png
  4. Screenshot 4 - Site Crawler Results
    Click Crawler tab, run a crawl, show the site overview with page scores.
    Save as: screenshot-4-crawler.png
  5. Screenshot 5 - Rank Checker
    Click Rank tab, check a keyword, show the position card and SERP results.
    Save as: screenshot-5-rank.png

Step 3: Quick alternative - Full page screenshot

  1. Open the side panel with analysis results showing
  2. Open DevTools, click the 3-dot menu > "Run command" (or Ctrl+Shift+P)
  3. Type screenshot and select "Capture full size screenshot"
  4. Crop to 1280x800 in any image editor

Tip: Window capture method

Alternatively, just use Win + Shift + S (Windows Snipping Tool) to capture the extension side panel while it's open next to a Shopify store. Resize to 1280x800.
